Computer format of the date and time consists of the numbers only, for example: 21.05.2018 16:30
Human prefer to see something like this: 21 May 2018 year, 16 hours 30 minutes
You task is simple - convert input date and time from computer format into "human" format

Input: Date and time as string

Output: The same date and time, but in more readable format

Example: 

checkio("01.01.2000 00:00") == "1 January 2000 year 0 hours 0 minutes"

How it is used: To improve the understanding between computers and humans.

Precondition:
0 < date <= 31
0 < month <= 12
0 < year <= 3000
0 < hours < 24
0 < minutes < 60
